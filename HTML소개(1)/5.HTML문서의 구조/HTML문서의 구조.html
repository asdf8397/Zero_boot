HTML 문서의 구조 - 도큐먼트 타입으로 HTML이다라고 말하는것 - html은 하나의 문서에서 하나만 있어야한다 그리고 모든 내용들이 태그내부에 자식요소로 들어가야한다. (최상위요소라고한다.) html: 페이지 전체의 컨텐츠를 감싸는 루트(root)요소 head태그는 화면에 보이지 않는 웹페이지의 정보들을 담고있다. 웹브라우저의 정보, 웹페이지의 검색엔진 눈에 보이지 않는 정보들 body태그는 웹브라우저가 화면에 나타나는 모든 컨텐츠 HTML문서의 구조 : 페이지 전체의 컨텐츠를 감싸는 루트(root)요소 : 웹브라우저 화면에 직접적으로 나타나진 않는 웹페이지의 정보 meta tag: 문서의 일반적인 정보와 문자 인코딩을 명시 title : 웹브라우저 화면에 나타나는 모든 컨텐츠
